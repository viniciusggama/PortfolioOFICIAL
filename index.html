Projetos em Destaque

De ideias abstratas a projetos bem-sucedidos, aqui estão alguns dos meus trabalhos mais recentes.

  • Plataforma de Cadastro para usuário

    Formulário de cadastro web que permite que um usuário se inscreva em um site ou serviço on-line fornecendo informações pessoais e de contato.

    Conhecer →

  • Ferramenta de verificação e registros de docentes

    O sistema foi desenvolvido para o Instituto Florence, com o objetivo de realizar o registro de docentes na faculdade via QR-CODE. O sistema inclui um PAINEL PRINCIPAL que exibe a imagem do docente, nome e localização, uma TELA ADMINISTRADOR para controle e relatórios, e uma TELA DO DOCENTE para login e leitura de QR-CODE.

  • Quiz Semana da Sustentabilidade

    O Quiz da Semana da Sustentabilidade foi desenvolvido para promover a conscientização ambiental entre os estudantes. Trata-se de um jogo interativo com perguntas sobre sustentabilidade, ajudando os participantes a aprender sobre práticas de consumo consciente e conservação de recursos.

    Conhecer →

  • Sistema de Cadastro de Periféricos

    O Sistema de Cadastro de Periféricos é uma solução para gerenciar e catalogar dispositivos periféricos de forma eficiente. Permite o registro detalhado dos equipamentos com informações cruciais como modelo, SO, usuário, quantidade de memória e localização. Com uma interface intuitiva, facilita a busca e consulta de dados, e oferece relatórios e análises para decisões.
